# ☕ Java Projects Collection

A curated collection of beginner-friendly **Java** and **JavaFX (FXML)** projects built for learning Object-Oriented Programming, GUI development, and software design principles.

This repository serves as a practice hub for students and aspiring developers who want hands-on experience with Java desktop application development using JavaFX, Scene Builder, and FXML.

---

## 📚 About

This repository contains multiple Java projects ranging from basic console applications to graphical desktop applications developed with JavaFX and FXML.

The goal of this repository is to:

- Learn Java programming fundamentals
- Practice Object-Oriented Programming (OOP)
- Build desktop GUI applications
- Understand JavaFX and FXML architecture
- Improve problem-solving and software development skills
- Create portfolio-ready projects

---
